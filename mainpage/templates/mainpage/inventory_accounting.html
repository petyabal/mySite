Приложение для учета материальных запасов в ООО "Норд Пилигрим"

Программный продукт разрабатывается в рамках шаблонного программного решения для курсовых и дипломных работ. В представленном примере инвентаризационный учет ведется в ООО «Норд Пилигрим», но в краткие сроки возможно изменение интерфейса и корректировка алгоритмов программного решения под условия другой организации. Разработка ведется в среде Microsoft Visual Studio 2015 Community Edition (язык C#). По вопросам приобритения ПП можно обращаться напрямую к разработчику.
Скачать программный продукт можно по этой ссылке

Системные требования:

  • ОС линейки Windows;
  • наличие предустановленного пакета .NET Framework 4.5.2
  • наличие предустановленной СУБД MySQL Server и инструмента для визуального проектирования баз данных MySQL Workbench;
  • база данных inventory_accounting, предустановленная на локальном сервере.

Скачать .NET Framework 4.5.2 можно с официального сайта Microsoft. Скачать MySQL Community Server, включающий в себя MySQL Workbench, можно с официального сайта проекта. Работоспособность приложения проверялась на Windows 7.

Порядок установки:

  • установить СУБД MySQL Community Server, и настроить ее (пароль qwerty для root аккаунта) с учетом строки подключения базы данных, испорльзующейся в программном продукте, и убедиться, что установленная версия поддерживает unicode кодировку (записи, содержащие кириллицу, сохраняются в БД);
  • импортировать базу данных inventory_accounting;
  • разместить файлы с программным продуктом в любой директории на компьютере, не изменяя расположение файлов и папок внутри папки данного программного решения.

Основы работы с программой:

Для авторизации в данном ПП используйте учетные данные приведенные в разделе ниже, в том случае, если вы еще не изменяли учетные данные пользователей средствами данного приложения либо средствами СУБД.

Пользователи, авторизовавшиеся в рамках учетной записи с типом "администратор" могут вносить изменения (создание, редактирование, удаление записей) в таблицу содержащую сведения о сотрудниках, и таблицу с учетными записями сотрудников. Весь остальной функционал ПП для них недоступен.

Пользователи с типом учетной записи "пользователь" выполняют саму задачу инвентаризационного учета в организации. Им доступен весь функционал приложения, за исключением функций управления учетными записями пользователей.

В главном меню приложения все функции разделены по разделам.

Раздел "Информационная база" предназначен для формирования актуальных сведений инвентаризационного учета в базе данных. Пункт меню "Организацинная информация" позволяет поддерживать актуальные сведения о сотрудниках предприятия, помещениях, видах и образцах (моделях) материальных запасов. Пункт меню "Материальные запасы" позволяет просматривать и изменять информацию о материальных запасах, находящихся на инвентаризационном учете в текущий момент времени. В пункте меню "Приходные накладные" можно формировать и просматривать сведения о всех поступлениях материалбных запасов. Пункт меню "Акты списания" позволяет учитывать в базе данных информацию о списании материальных запасов.

При дальнейшей разработке планируется доработка существующих алгоритмов, добавление программного формирования документов и создание интерфейса для изменения логинов и паролей учетных записей самими пользователями.

Строка подключения БД:

DataBase = inventory_accounting; Data Source = localhost; userID = root; password = qwerty
