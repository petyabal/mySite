Приложение для формирования и ведения графика отпусков сотрудников "Колэнерго"

Приложение, предназначенное для автоматизации процесса формирования и ведения графика отпусков сотрудников ПАО "МРСК Северо-Запада" ("Колэнерго") создавалось в качестве дипломной работы в апреле-июне 2018 года. Разработка велась в среде Microsof Visual Studio 2010 Express (язык C#).
Скачать программный продукт и сопутствующую документацию можно по этой ссылке

Системные требования:

  • ОС линейки Windows;
  • предустановленный радактор электронных таблиц Microsoft Excel 2010;
  • наличие предустановленных пакетов .NET Framework 4 и .NET Framework 4.5.2 (для установки MySQL Workbench);
  • наличие предустановленной СУБД MySQL Server и инструмента для визуального проектирования баз данных MySQL Workbench;
  • база данных vacation_schedule, предустановленная на локальном сервере.

Скачать .NET Framework 4 и .NET Framework 4.5.2 можно с официального сайта Microsoft. Скачать MySQL Community Server, включающий в себя MySQL Workbench, можно с официального сайта проекта. Работоспособность приложения проверялась на Windows 7.
