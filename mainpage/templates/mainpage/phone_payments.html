Программа для учета платежей за услуги мобильной связи

Приложение разрабатывалось в рамках курсовой работы с применением технологии типизированного файла в среде разработки Delphi 7 (язык Object Pascal) в мае 2017 года.
Скачать программный продукт можно по этой ссылке

Системные требования:

  • ОС линейки Windows;
  • наличие предустановленного текстового редактора Microsoft Word.

Работоспособность приложения проверялась на Windows 7.

Порядок установки:

Распаковать архив с программой в отдельную папку. Папку с разархивированными файлами можно разместить в любой директории на ПК или съемном носителе информации. Нельзя удалять, перемещать или изменять файлы из папки.

Основы работы с программой:

Доступ к функциям программы защищен паролем. Для пользователей без прав администратора доступно 3 попытки ввода пароля. После исчерпания числа попыток ввода пароля учетная запись пользователя блокируется и разблокировать ее может только пользователь с правами администратора. Ограничение числа попыток ввода пароля у пользователя с правами администратора не предусмотрено, но после каждой попытки авторизоваться, если был введен неправильный пароль, возможность ввода нового пароля блокируется на 30 секунд.

Функционал у пользователя с правами администратора более широкий, т.к. кроме функционала доступного для всех пользователей, также доступна возможность управления учетными записями пользователей. Недоступна для администратора только возможность изменить пароль собственной учетной записи.

Типизированный файл, содержащий сведения о совершенных транзакциях создается один раз при проведении самой первой транзакции в начале рабочего дня и содержит в названии текущую дату. При всех последующих обращениях в этот день данные дозаписываются в созданный файл. Доступна функция импорта сведений из типизированного файла в текстовый редактор Microsoft Word.
